# Academia Connect (Team 01)

Project Overview

Academia Connect is a web platform connecting University Students/Faculty with the Industrial Sector. Built for CSAI 203 (Introduction to Software Engineering).

Setup Instructions

1.Clone the repository.

2.Copy config_template.py to config.py and update the Server Name.

3.Install dependencies: pip install -r requirements.txt

4.Run the app: python run.py

Structure

1.app/: Source code (MVC Pattern)

2.docs/: Project Documentation (SRS, SDD)

3.tests/: Automated Unit Tests
